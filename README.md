# SwiftUI MVVM COREDATA NOTE APP 
This a simple swiftui app where i used mvvm architecture, coredata swiftui and so on...


#FEATURES
1. SWIFTUI
2. MVVM
3. COREDATA
4. NAVIGATIONLINK
5. SHEETVIEW
